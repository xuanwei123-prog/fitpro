# 🏃‍♂️ FitPro - Health & Fitness Management System

FitPro is a robust desktop application designed to help users effectively manage their health, track fitness goals, and monitor nutritional intake. Developed using **Java** and **JavaFX**, this application provides an intuitive graphical user interface for a seamless user experience.

## ✨ Key Features

* **User Authentication & Profile:** Secure login system with customizable user profiles.
* **Goal Management:** Set, track, and update personalized fitness and health goals.
* **Nutrition Tracking:** Log daily meals and monitor nutritional data.
* **Exercise & Log Management:** Keep detailed records of daily physical activities.
* **Device Integration:** Interface for managing and syncing external fitness tracking devices.

## 🛠️ Tech Stack

* **Language:** Java
* **GUI Framework:** JavaFX
* **Build Tool:** Maven (pom.xml included)
* **Architecture:** MVC (Model-View-Controller) pattern implemented for clean code separation.
